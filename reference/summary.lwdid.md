# Summarise an lwdid object

Calls
[`print.lwdid()`](https://xiangao.github.io/lwdidR/reference/print.lwdid.md)
and additionally prints period-specific ATTs (common-timing design) or
the first 10 (g, r)-specific ATTs (staggered design).

## Usage

``` r
# S3 method for class 'lwdid'
summary(object, ...)
```

## Arguments

- object:

  An object of class `"lwdid"` as returned by
  [`lwdid()`](https://xiangao.github.io/lwdidR/reference/lwdid.md).

- ...:

  Currently ignored.

## Value

Invisibly returns `object`.
